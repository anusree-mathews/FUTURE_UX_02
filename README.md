Glam Beauty Salon – UI/UX Design Project 💄✨

Overview

Glam Beauty Salon is a mobile-first UI/UX design project created to provide a seamless and elegant salon booking experience. The design focuses on simplifying service discovery, appointment scheduling, and customer engagement through an intuitive and visually appealing interface.

Project Goal

The primary goal of this project is to create a user-friendly salon application that allows customers to:

- Browse beauty services
- View service details and pricing
- Schedule appointments easily
- Receive booking confirmation
- Enjoy a smooth and aesthetically pleasing user experience

Features

🔐 Login Screen

- Clean and elegant user authentication interface
- Username and password input fields
- Sign-up and password recovery options

💇 Service Browsing

- Categorized services:
  - Hair Services
  - Skin Treatments
  - Makeup Services
- Search functionality for quick service discovery
- Service cards with images and details

🎨 Service Details

- Detailed service information
- Pricing display
- High-quality service images
- Quick booking option

📅 Appointment Booking

- Interactive date selection
- Time slot selection
- Simple booking workflow
- User-friendly appointment scheduling experience

✅ Booking Confirmation

- Appointment summary
- Service details
- Date and time confirmation
- Payment status information

Design Highlights

- Soft pink luxury-inspired color palette
- Modern and clean interface
- Mobile-first design approach
- User-centered navigation
- Consistent visual hierarchy
- Conversion-focused booking flow

Tools Used

- Figma
- UI/UX Design Principles
- Wireframing & Prototyping

Screens Included

1. Login Page
2. Services Page
3. Service Details Page
4. Appointment Booking Page
5. Booking Confirmation Page

UX Design Decisions

- Simplified navigation to reduce user effort
- Prominent booking CTAs to improve conversions
- Visual service categorization for better discoverability
- Minimal form fields to streamline appointment booking
- Consistent layout and branding across all screens
